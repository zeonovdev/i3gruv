# i3wm gruvbox dark setup

My i3wm config in gruvbox dark theme.

# Installation
1. Clone repo:
    ```bash
    git clone https://github.com/zeonovdev/i3gruv.git
    cd i3gruv
    ```

2. Start installation script:
    ```bash
    chmod +x setup.sh
    ./setup.sh
    ```
3. Reboot system.

### Apps 
App/power/wifi menu: rofi
Terminal: kitty
Bar: polybar
Compositor: picom
